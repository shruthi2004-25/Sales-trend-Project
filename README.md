# Sales Trend Visualization and Forecasting Project

## Project Description
This project analyzes sales data using Python and performs data visualization, exploratory data analysis, and forecasting using machine learning techniques. It helps to understand sales trends and predict future sales.

## Objectives
- To analyze sales data
- To visualize sales trends
- To identify top products and regions
- To predict future sales using machine learning

## Dataset Used
Superstore Sales Dataset  
https://kaggle.com

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Machine Learning Model
6. Sales Forecasting

## Features
- Monthly sales trend analysis
- Profit analysis
- Region-wise sales analysis
- Top product analysis
- Sales forecasting using Linear Regression

## Results
The project successfully visualizes sales trends and predicts future sales using machine learning.

## How to Run
1. Open Google Colab
2. Upload dataset
3. Run notebook step by step
